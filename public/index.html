How to use (Kaggle + 100Labs)

We use free Kaggle resources to run the open‑source Chatterbox TTS from MIT‑licensed authors, and 100Labs is a lightweight Windows UI that connects to it.

Step 1 — Create a Kaggle notebook

  1. Sign up or sign in to Kaggle.
  2. Create a new Notebook.

Step 2 — Paste and run this cell

Copy the following and run in the first cell:

!git clone https://github.com/resemble-ai/chatterbox.git
%cd chatterbox
!pip install chatterbox-tts
!pip install protobuf==4.25.3 google-api-core==2.19.1 google-cloud-bigquery==3.31.0 rich==13.7.1 gradio
!python gradio_tts_app.py

Wait until the cell finishes. It will display a public Gradio URL.

Step 3 — Download and run 100Labs

  1. Download 100labs.exe from this site.
  2. Open it and paste the Gradio URL from Kaggle into the app.
  3. Start generating speech: choose voice, clone speaker, and synthesize.

Why this approach?

Commercial TTS like ElevenLabs can be expensive. 100Labs provides a free, accessible UI on Windows that talks to the open‑source Chatterbox model via Kaggle’s compute. It’s a practical workaround for people without GPUs or budget—yet still delivers high‑quality voices and cloning.

System requirements

Windows 10/11, 4 GB RAM, 500 MB free space.
